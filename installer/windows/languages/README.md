# Inno Setup Language Asset

`ChineseSimplified.isl` is vendored from Inno Setup tag `is-6_3_2`
(peeled commit `617e432f0b22aa148e7152acf88a18e7586c2c88`) at:

https://github.com/jrsoftware/issrc/blob/is-6_3_2/Files/Languages/Unofficial/ChineseSimplified.isl

The repository copy is normalized to UTF-8 with LF line endings. Its SHA-256
is `abb4c64bc7a03f5db7ab16fdd88d5e14e133426cc8a0c48e21149de8b40967b0`.

The Chocolatey Inno Setup 6.3.2 package does not install unofficial language
files, so the Windows installer must use this pinned local copy.
